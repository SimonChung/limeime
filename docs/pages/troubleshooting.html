疑難排解

無法啟用或叫不出鍵盤 打不出字 資料庫問題 語音輸入

打不出字、找不到輸入法或備份還原出問題時，可依下方常見狀況快速排查。遇到問題時，先回到「設定」分頁看狀態橫幅的顏色，多數啟用相關問題都能由此判斷。

無法啟用或叫不出鍵盤

  • 狀態橫幅是紅色：代表尚未啟用萊姆輸入法鍵盤，請回到「快速上手」依平台完成啟用。
  • 狀態橫幅是橙色（iPhone）：代表鍵盤已啟用，但尚未允許完整取用。若需要按鍵震動，請開啟「允許完整取用」，不需要震動則可忽略，仍能打字。
  • 叫出的不是萊姆輸入法：在任一輸入欄位長按或點按系統的地球鍵，切換到萊姆輸入法。

打字沒有候選字、打不出字

  • 確認已安裝輸入法：到「輸入法」分頁確認至少有一套輸入法且已啟用。若清單為空，請點「＋」下載或匯入，詳見「管理輸入法」。
  • 確認輸入模式：確認目前是中文模式，必要時點「中／英」鍵切換。
  • 剛清除過輸入法：若先前移除了正在使用的輸入法，鍵盤會找不到候選字，請重新下載或切換到其他已安裝的輸入法。
i
找不到某個字可使用「字根反查」功能，在喜好設定的「字根反查設定」指定反查來源，打字找不到字時以其他輸入法字根標注說明。

備份還原相關問題

  • 還原後資料不見了：還原會以備份檔完全取代現有資料。若還原後內容不如預期，請改用正確的備份檔重新還原。
  • 備份卡住：大型資料庫打包需要數秒到數十秒，畫面會顯示進度或「準備備份中…」，請耐心等待，勿中途關閉 App。
  • 資料庫疑似損壞：可嘗試「還原預設資料庫」回到初始狀態，再重新下載需要的輸入法。此操作會覆蓋現有資料。

語音輸入

語音輸入行為依平台而不同。

  • Android：可使用系統或 Google 語音輸入。萊姆輸入法內建的語音輸入需要麥克風（RECORD_AUDIO）權限，若略過或拒絕，仍可改用 Google／廠商的語音輸入。
  • 權限被永久拒絕：若系統不再跳出授權視窗，請前往系統設定，於萊姆輸入法的「權限 → 麥克風」改為允許。
i
仍無法解決？歡迎到 GitHub Issues 回報問題，附上裝置型號與系統版本，有助於更快定位。
