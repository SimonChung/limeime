不收集任何個人資料

萊姆輸入法不蒐集、不上傳你輸入的任何文字。完整取用權限僅用於按鍵震動回饋，可自由選擇開啟。

完全開源

所有程式碼公開在 GitHub，任何人都能檢視、回報問題與貢獻，也可自行取得原始碼建置。

打字習慣，跨平台延續

透過備份與還原，將輸入法、碼表與學習記錄跨平台轉移，在 Android、iPhone 與 iPad 之間延續慣用的打字手感。

Features

為中文輸入打造的細節

關聯字學習、字根反查、簡繁轉換到鍵盤外觀，每個設定都為了讓你打得更快、更順手。

關聯字與智慧學習

依你的輸入習慣自動學習新詞與關聯字，常用字自動往前排，越打越懂你。

自動學習選取排序關聯字庫

字根反查

打字找不到字時，用其他輸入法的字根即時標注說明，倉頡、注音互相對照不卡關。
